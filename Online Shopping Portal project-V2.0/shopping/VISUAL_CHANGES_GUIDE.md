## ✅ **INTERACTIVE FEATURES YOU'LL NOTICE**

### **Button Interactions:**
1. **Hover Over Any Button**
   - Button lifts up (moves up 2 pixels)
   - Shadow becomes darker/larger
   - Smooth animation (0.3 seconds)

2. **Click a Button**
   - Button returns to normal position
   - Visual feedback is immediate
   - Smooth transition

### **Form Interactions:**
1. **Click on an Input Field**
   - Border changes from gray to **purple**
   - A soft **purple glow** appears around it
   - Smooth focus transition

2. **Type in Field**
   - Border stays purple
   - Rounded corners (8px radius)
   - Clean, modern appearance

3. **Leave the Field**
   - Border returns to gray
   - Glow disappears
   - Smooth unfocus transition

### **Card Interactions:**
1. **Hover Over a Card**
   - Card lifts up
   - Shadow increases
   - Smooth animation

2. **Move Away**
   - Card returns to position
   - Shadow returns to normal
